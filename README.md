# FashionStore (Vastra)

A full-stack E-Commerce web application developed using Java Enterprise technologies.

## Technologies Used

- Java
- JDBC
- Servlets
- JSP
- MySQL
- Hibernate
- Maven
- Apache Tomcat

## Features

- User Registration and Login
- Product Catalog
- Category Management
- Shopping Cart
- Order Management
- Database Integration
- Responsive UI

## Project Structure

- MVC Architecture
- DAO Layer
- Model Layer
- Controller Layer
- JSP Views

## Tools Used

- Eclipse IDE
- Apache Tomcat
- MySQL Workbench
- Git & GitHub
